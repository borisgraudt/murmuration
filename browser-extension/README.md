# Murmuration Chrome Extension

Chrome extension для поддержки `mur://` протокола.

## Установка

1. Открой Chrome и перейди в `chrome://extensions/`
2. Включи "Режим разработчика" (Developer mode)
3. Нажми "Загрузить распакованное расширение" (Load unpacked)
4. Выбери папку `browser-extension/`

## Использование

- Клики по `mur://` ссылкам автоматически открывают контент через Web Gateway
- Навигация на `mur://` URLs перехватывается и перенаправляется
- Адресная строка показывает оригинальный `mur://` URL (насколько возможно)

## Настройки

Нажми на иконку расширения чтобы настроить порт Web Gateway (по умолчанию 17081).

## Примечания

- Chrome не позволяет полностью изменить протокол в адресной строке из соображений безопасности
- Расширение перехватывает `mur://` и перенаправляет на Web Gateway
- Title страницы обновляется на `mur://` URL
- История браузера хранит оригинальный `mur://` URL в state
